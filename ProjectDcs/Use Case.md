# Use Cases - ksf_Teams

## UC-TE-001: Create Team
**Actor**: HR Manager

**Flow**:
1. Navigate to Teams > New
2. Enter team name, description
3. Assign team lead
4. Add members (from ksf_HRM)
5. Save

## UC-TE-002: Add Employee to Team
**Actor**: HR Manager

**Flow**:
1. Open team
2. Click "Add Member"
3. Search and select employee
4. Set role in team
5. Save
